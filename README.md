## About Oh Dear CLI

Oh Dear CLI was created by, and is maintained by [Nuno Maduro](https://github.com/nunomaduro), and is a Oh Dear CLI tool written in PHP with Laravel Zero.

## Installation

> **Requires [PHP 7.1.3+](https://php.net/releases/)**
